Everyone is welcome to contribute to Viseron. The easiest way is to click the ‚≠ê button above!

Some of the ways you can contribute are:

- [Reporting bugs](https://github.com/roflcoopter/viseron/issues/new?labels=bug)
- [Requesting features](https://github.com/roflcoopter/viseron/issues/new?labels=enhancement)
- Answering questions in [Issues](https://github.com/roflcoopter/viseron/issues) or [Discussions](https://github.com/roflcoopter/viseron/discussions)
- Improving the documentation
- Star the project!

## Developers

I will gladly accept any pull request that is opened, as long as it follows the [Style guidelines](/docs/developers/development_environment/style_guidelines).
If you want to help out with the development of Viseron, head over to the [Developer documentation](/docs/developers) to get started.
